## 세팅  
  
우선 세팅은 이렇게 해놨다.  
옛날에는 별의별거 다 깔아서 사용하고 그랬는데...  
이제는 귀찮아서 이렇게만 쓴다.  
  
현재는 맥 기본터미널에 vim 조합이 아닌 Terminus에 nvim을 쓰고있다.  
로컬도 Terminus를 사용한다.

```sh
call plug#begin('~/.vim/plugged')

Plug 'EdenEast/nightfox.nvim'
Plug 'preservim/nerdtree'
Plug 'vim-airline/vim-airline'
Plug 'neoclide/coc.nvim', {'branch': 'release'}

call plug#end()

colorscheme nightfox

nmap <C-T> : NERDTreeToggle<CR>
nmap <leader>a  <Plug>(coc-codeaction-selected)

" vim config
set ts=4
set shiftwidth=4
set autoindent
set cindent
set nu
set hlsearch
set showmatch
set noswapfile
set ruler
set laststatus=2
```

## 플러그인

<img src="/assets/images/posts/etc/nvim.png" width="100%" height="50%">  
  
### NerdTree  
  
Vim을 좋아하고 많이 사용하는 사람들 중에서 이거 모르는 사람은 없을 것 같다.  
위에 세팅을 보면 Ctrl-T로 맵핑을 했는데 해피해킹을 사용하면 캡스락 키에 컨트롤 키가 맵핑되어있어 저 조합이 아주 편하다.  
(지금은 CIDOO사의 QK61을 해피해킹처럼 맵핑해서 쓴다)  
[preservim/nerdtree](https://github.com/preservim/nerdtree.git)
  
### EdenEast/nightfox.nvim  
  
위에 사진을 보면 파란것도 아니고 검은것도 아닌 저 바탕색이 이 테마를 사용해서 그렇다.  
찾아보면 많은 테마가 있는데 유독 저 테마가 내 관심을 끌었고 근 2년정도 저 테마만 사용중이다.  
이전에 사용하던 테마가 jellybeans였나.. 그랬던걸로 기억하는데 그 테마가 개인적으로 너무 쨍한 느낌이라 싫어서 바꿨다.  
[EdenEast/nightfox.nvim](https://github.com/EdenEast/nightfox.nvim.git)  
  
### vim-airline/vim-airline
  
Vim화면 하단에 파일명이나 경로, 버퍼 등 추가 정보를 표기해주는 플러그인이다.  
뭐 편하다고 해서 설치는 해봤는데... 지금은 그냥 없으면 밋밋해서 설치해두는정도...  
그냥 색이 이뻐서 강추다 ㅋ  
[vim-airline](https://github.com/vim-airline/vim-airline.git)
  
### neoclide/coc.nvim  
  
아 이거 아주 좋다. 이건 꼭 써야한다.  
다른 언어는 모르겠는데 자바스크립트, 타입스크립트 한정으로는 나는 아주 괜찮게 쓴 플러그인이다.  
자동완성 플러그인인데 처음에 사용법이 약간 아리송하다. 나는 지금 Ctrl-P로 세팅되어있는데  
추천 완성 탭이 켜지면 방향키나 탭 키로 선택하는 설정이 있을텐데 그걸 못찾아서 그냥 기본세팅으로 쓴다. 그래도 편하니까 아주 강추...  
[neoclide/coc.nvim](https://github.com/neoclide/coc.nvim.git)
